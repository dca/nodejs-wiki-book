# Ubuntu Linux

## 使用 nvm 安裝 (推薦)

```
git clone git://github.com/creationix/nvm.git ~/.nvm
echo ". ~/.nvm/nvm.sh" >> ~/.bashrc
nvm install v0.12.0
nvm alias default v0.12.0
```

以上可參考： https://github.com/creationix/nvm/
